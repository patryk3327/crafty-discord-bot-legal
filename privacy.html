Your Bot Name – Privacy Policy

Last updated: DATE

1. Overview

This Privacy Policy explains what information Your Bot Name (“the bot”) processes when you use it in your Discord server and how that information is used.

2. Information the Bot Processes

To function, the bot may temporarily process:

  • Discord server ID and channel ID (to know where to respond)
  • User ID of the person using commands (to handle permissions and replies)
  • Command content (e.g., which server to query or manage)

This information is used only to execute the requested command and provide responses such as server status or management results.

3. Data Storage

The bot does not intentionally store personal data from Discord users long-term, except for minimal configuration data if required (for example, which channel to post status updates in). Any such configuration data is limited to what is necessary for the bot to operate.

4. Crafty Controller Data

The bot connects to your Crafty Controller instance to read server status and send management commands. Any data about your servers (such as server name, online/offline status, or player count) is obtained from Crafty Controller and is used only to display or act on that information in Discord.

5. Third Parties

The bot does not share your data with third parties. However, it runs on infrastructure provided by hosting providers (for example, a VPS or hosting platform), which may process technical data such as IP addresses and logs as part of normal server operation.

6. Data Security

Reasonable measures are taken to protect configuration data and access credentials (such as the Crafty Controller API or URL) from unauthorized access. You are responsible for keeping any tokens, API keys, or URLs you configure for the bot secure.

7. Data Deletion

If you want any stored configuration data related to your server to be removed, you can remove the bot from your server and contact the developer to request deletion of any remaining data.

8. Changes to This Policy

This Privacy Policy may be updated from time to time. Continued use of the bot after changes are published means you accept the updated policy.
